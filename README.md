# SpellBuddy (Web)

A beginner-friendly starter for a dyslexia-smart spelling & vocabulary practice app that runs great on iPad.

## Quickstart
1. Install Node.js LTS (v20+), Git, and VS Code.
2. In a terminal:
   ```bash
   npm install
   npm run dev
   ```
3. Open http://localhost:3000

## Structure
- `app/` — Next.js App Router pages
- `components/` — reusable UI
- `lib/` — tiny helpers (localStorage etc.)
- `public/` — static assets

## What’s included
- Parent Dashboard to paste weekly words
- Local-only storage (safe to play)
- Study screen with dictation (text-to-speech) and typed answer
- Dyslexia-friendly defaults (Lexend font, big targets)
